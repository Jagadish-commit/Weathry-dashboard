# Weathry 🌦️

Weathry is a modern weather dashboard application built with React.js and Tailwind CSS. It provides real-time weather updates, air quality index (AQI), forecasts, and interactive temperature visualization with a clean responsive UI.

## Features

- Real-time weather updates
- 5-day weather forecast
- Air Quality Index (AQI)
- Temperature charts
- Responsive design
- Multi-language support
- Dynamic weather icons

## Tech Stack

- React.js
- Tailwind CSS
- Vite
- OpenWeather API
- Chart.js
